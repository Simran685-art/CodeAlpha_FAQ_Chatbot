# 🤖 FAQ Chatbot

## 📌 Project Description

FAQ Chatbot is an AI-inspired question-answering application developed using Python and Streamlit. The chatbot uses Natural Language Processing (NLP) techniques to find the most relevant answer from a predefined knowledge base.

## 🚀 Features

- Interactive FAQ Chatbot
- NLP-based Question Matching
- Confidence Score Display
- Chat History
- Clear History Option
- FAQ Categories Sidebar
- Unknown Question Handling
- User-Friendly Interface
- Real-Time Responses

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-Learn
- TF-IDF Vectorizer
- Cosine Similarity

Install dependencies:

```bash
streamlit run app.py
```
